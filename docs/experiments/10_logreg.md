# ロジスティック回帰（Logistic Regression）

## このノートブックでやること

シグモイド関数と交差エントロピーを使った最もシンプルな分類モデルで、タイタニックの生存予測を行う。

## アルゴリズム解説

### ゼロつく1で学んだことの「単純版」

ゼロつく1で出てきた **シグモイド関数** を覚えているだろうか：

```
σ(x) = 1 / (1 + exp(-x))
```

ニューラルネットでは、これを隠れ層の活性化関数として使った。
ロジスティック回帰は、**隠れ層なしで直接シグモイド関数を適用する** モデルだ。

```
[ニューラルネット]
  入力 → 隠れ層(ReLU) → 隠れ層(ReLU) → 出力層(sigmoid) → 確率

[ロジスティック回帰]
  入力 → sigmoid → 確率
```

つまり、ロジスティック回帰 = **ニューラルネットの最小構成（層が1つだけ）** と言える。

### 予測の流れ

```
Step 1: 重み付き和を計算
  z = w1×pclass + w2×sex + w3×age + ... + b

Step 2: シグモイドで確率に変換
  p = σ(z) = 1 / (1 + exp(-z))

Step 3: 閾値で分類
  予測 = 1 (生存) if p ≥ 0.5 else 0 (死亡)
```

### 学習の仕組み

損失関数は **交差エントロピー**（ゼロつく1の多クラス分類で使ったものの2クラス版）：

```
Loss = -[y·log(p) + (1-y)·log(1-p)]
```

これを勾配降下法で最小化して、重み w を最適化する。
ゼロつく1で学んだ逆伝播と本質的に同じ仕組みだ。

### StandardScaler が必要な理由

ロジスティック回帰は入力の重み付き和を計算するので、**特徴量のスケールが揃っていないと正しく学習できない**。

```
age: 0〜80     → 重みが小さくなる
fare: 0〜512   → 重みがさらに小さくなる
sex: 0 or 1    → 重みが大きくなる
```

StandardScaler で平均0・標準偏差1に揃えることで、全特徴量が公平に扱われる。

### 正則化（C パラメータ）

```
C = 1.0（デフォルト）
→ C が小さいほど正則化が強い（重みを小さく保つ）
→ C が大きいほど正則化が弱い（データに忠実にフィット）
```

正則化はゼロつく1の **Weight Decay** に相当する。過学習を防ぐための仕組みだ。

## ノートブックの流れ

1. **Setup**: ライブラリ読み込み、設定ファイルの読み込み
2. **Data Loading**: v1特徴量パイプラインで14特徴量を生成
3. **Feature Engineering**: 使用する14特徴量の確認
4. **Cross-Validation**: 5-fold StratifiedKFold で評価
5. **Train Full Model**: 全学習データで再学習、テストデータを予測
6. **Submission**: 提出ファイル生成
7. **Analysis**: 回帰係数のバーチャートで「どの特徴量が生存に効くか」を可視化

## 実験結果

> **交差検証（CV）って何？** → [01_preprocess/README.md](01_preprocess.md) で詳しく解説している。
> 簡単に言えば、データの分割を5回変えてそれぞれ評価し、平均をとる手法だ。
> 1回きりの評価より信頼できるスコアが得られる。

### 係数の解釈

回帰係数が大きい特徴量ほど生存への影響が強い。
sex（性別）が最も大きな正の係数を持ち、pclass（客室クラス）が負の係数を持つ。

## ポイント・学び

- **ロジスティック回帰はNNの基本パーツ**。ゼロつく1で学んだ sigmoid + 交差エントロピーの「最小構成」がこのモデルだ
- **係数を直接解釈できる**のがロジスティック回帰の大きな強み。どの特徴量が効いているかが一目でわかる
- **StandardScaler は線形モデルの必需品**。決定木系のモデルでは不要だが、ロジスティック回帰では必須だ
- **AUC=0.8536** は「この問題にしては良好」。シンプルなモデルでもデータの特徴を適切に捉えている
